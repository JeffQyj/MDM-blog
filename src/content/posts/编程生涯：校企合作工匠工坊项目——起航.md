## 引言：不一样的课堂

2024年10月9日，一个平凡的秋日早晨。

但这一天，对我来说，却有些不同。

教室里来了几位「不速之客」——他们不是学校的老师，而是来自江苏一道云科技发展有限公司的工程师。西装革履的面孔背后，是一个个在企业里摸爬滚打多年的开发者。他们带来的，是一个叫做「工匠工坊」的校企合作项目。

说实话，在那之前，我对「校企合作」这四个字并没有太多感觉。总觉得不过是学校和企业签个协议、走个过场、请几个工程师来讲两节课罢了。

但这次，似乎不太一样。

工程师们没有照本宣科。他们带来的是真实的企业开发案例，讲的是真正在用的技术栈，甚至布置的作业都是实际项目中会遇到的场景。「13周时间，」为首的工程师说，「让你们体验一下什么叫真正的企业开发。」

那一刻，我便隐约觉得，这趟旅程可能会比我想象的要精彩得多。

## 项目全貌：工匠工坊是什么

### 核心理念

「工匠工坊」这个名称本身就透露了项目的定位——它不是普通的课外兴趣班，而是一套**完整的企业开发流程模拟**。

在这个项目里，我们不是学生，而是「实习生」。我们不承担KPI，但需要按时完成任务；我们不是正式员工，但同样要参与代码评审和项目复盘。工程师们带来的不是教科书式的知识，而是企业在实际生产环境中沉淀下来的开发规范与协作经验。

### 技术选型：为什么是 uniapp

项目选择了 **uniapp** 作为核心技术框架。这个选择并非随意为之，而是经过深思熟虑的：

| 维度       | 考量                        |
| -------- | ------------------------- |
| **学习曲线** | 基于 Vue.js，对有前端基础的同学友好     |
| **多端适配** | 一套代码，可发布到微信、支付宝、H5等多端     |
| **企业需求** | 国内小程序开发生态成熟，就业市场广阔        |
| **开发工具** | HBuilderX 提供完整的IDE支持，开箱即用 |

uniapp 由 DCloud 推出，基于 Vue.js 语法，能够实现「一次开发，多端运行」。据官方数据显示，截至2025年，uniapp 已成为国内跨平台开发领域最受欢迎的框架之一，被广泛应用于企业级应用、电商平台、社交软件等场景。

### 阶段划分：13周的旅程

整个项目分为四个阶段，层层递进：

| 阶段       | 时间     | 主题   | 核心目标                     |
| -------- | ------ | ---- | ------------------------ |
| **第一阶段** | 第1-4周  | 框架入门 | uniapp基础语法、HBuilderX工具掌握 |
| **第二阶段** | 第5-8周  | 独立实战 | 独立完成项目、接口联调、版本迭代         |
| **第三阶段** | 第9-12周 | 团队协作 | 小组分工、协作开发、完整项目交付         |
| **收尾阶段** | 第13周   | 总结复盘 | 项目复盘、经验沉淀、成果展示           |

「老师教学→独立demo实现→小组分工协作」，这十二个字概括了整个项目的学习路径。从理论到实践，从个人到团队，每一步都有明确的目标和可交付的成果。

## 企业导师：他们带来了什么

### 江苏一道云科技发展有限公司

该公司是一家专注于小程序开发和企业信息化服务的科技公司。在这次合作中，企业派出了具备丰富实战经验的工程师团队，为我们提供全程技术指导。

### 教学模式：从「灌知识」到「带着做」

与企业导师相处的这段时间，给我印象最深的不是某一行代码，而是他们授课的方式。

传统教学模式是「老师讲、学生听」，知识点是碎片化的，学生往往「一听就会、一做就废」。而企业导师采用的是「带着做」的模式，与前面的阶段划分完全对应：

- **第一阶段（第1-4周）→ 框架入门**：集中教学，系统讲解 uniapp 框架
- **第二阶段（第5-8周）→ 独立实战**：在老师指导下独立完成简单项目
- **第三阶段（第9-12周）→ 团队协作**：分组分工，开始模拟企业开发流程
- **收尾阶段（第13周）→ 总结复盘**：项目收尾、复盘总结、成果展示

这种模式让学习不再是空中楼阁。每一个知识点，都对应着可运行的代码；每一项技能，都在项目中得到了验证。

## 第一天：人生的第一次项目实训

从那天开始，我正式进入了人生的第一次项目实训。

2024年10月9日，企业导师站在讲台上，用投影仪展示着真实的项目代码；我们在台下，带着好奇和些许忐忑，听完了关于 uniapp 的第一堂课。

当时的我们并不知道后面的旅程会是怎么样的，但我们都知道，这一定会是一段精彩的旅程。
