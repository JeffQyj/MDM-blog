## 站在终点的回望

2023年12月31日，又一年走到尽头。

如果要用一个词形容我的2023年，大概是「突破」。从JavaWeb到SSM框架，从Vue到React，从虚拟机到Docker——这一年的变化，比我预想的要剧烈得多。

:::note
这篇文章不求面面俱到，只想记录那些真正塑造了我技术认知的瞬间。
:::

## 技术之路：从「会用」到「理解」

如果说2022年是「编程入门」，那2023年就是「理解工程」。

### 全栈的起点：Java与OOP

二月份，我开始学习Java。那段时间最大的困惑是：「为什么要这么繁琐？」

创建一个输入要new Scanner对象，定义一个类要把方法和属性放在一起，还要什么构造方法、getter/setter……相比之前学的C语言，Java看起来绕了好大一圈。

直到有一天，我终于想通了。

> **类是什么？类就是把数据和操作数据的方法打包在一起。**
> **private是什么？是「封装」，是保护，是把不该让别人直接访问的东西藏起来。**

就像银行的柜台——你不能直接进金库拿钱，必须通过柜员，柜员会核验你的身份、校验金额是否合法，然后才会帮你操作。

**OOP思想是真的优雅。**

### 数据库：从CRUD到原理

四月份学习MySQL，一开始觉得SQL语法太友好了——SELECT * FROM users，多清晰啊，比Java那些层层嵌套的调用链简洁多了。

后来才发现，数据库的水深着呢。

索引是怎么工作的？B+树的叶子节点为什么用链表连接？事务的隔离级别是怎么回事？MVCC是怎么实现快照读的？

> **越学，越觉得自己无知。**

这种感觉，大概就是编程学习路上的「常态」。每掌握一个知识点，就会发现这个知识点背后还有更深的一层。

### SSM框架：配置地狱与豁然开朗

十月份学习SSM框架（Spring+Spring MVC+MyBatis），那配置量让我一度怀疑人生。

- applicationContext.xml
- spring-mvc.xml
- web.xml
- db.properties
- Mapper.xml

整整一套XML配置下来，我终于理解了什么叫「框架整合」。

但后来学Spring Boot的时候，当我发现只需要一个application.yml就能搞定一切，我才知道什么叫「站在巨人的肩膀上」。

> **没有SSM的「手动配置」经历，就不会理解Spring Boot的「自动配置」有多爽。**

### 前端框架：从Vue到React

十二月份从Vue转向React，最大的冲击是JSX。

Vue的模板语法让我感觉很亲切——写起来像写HTML。但React的JSX完全不同——在JavaScript里写HTML？

```jsx
return (
    <div>
      <h1>{title}</h1>
      <button onClick={handleClick}>点击</button>
    </div>
);
```

一开始满脑子问号，但后来我明白了：**Vue的模板是「HTML扩展」，React的JSX是「JavaScript扩展」**。两者思路完全相反，但殊途同归——最终都是生成DOM。

学完React之后，我最大的感悟是：

> **框架是工具，思维才是核心。**

无论是Vue的「响应式数据驱动」，还是React的「状态驱动渲染」，它们解决的都是同一个问题：如何让UI和数据保持同步。

### 容器化：从虚拟机到Docker

十二月份学习Docker和容器化技术，最大的收获是对「隔离」这个概念的理解。

虚拟机隔离的是硬件，容器隔离的是进程。虚拟机需要虚拟化整个操作系统，容器只需要隔离进程——所以容器比虚拟机轻量得多。

> **隔离才是问题，环境不应该是问题。**

## 成长碎记：那些塑造我的瞬间

### 一月：万物底层逻辑相通

寒假的某一天，我站在灶台前突然发现：做菜和编程，竟是同一套逻辑。

- 做菜：备料 → 调味 → 火候 → 成品 → 复盘
- 编程：需求 → 设计 → 实现 → 测试 → 迭代
- 做人：模仿 → 思考 → 行动 → 反思 → 成长

**我们以为的「跨界」，其实底层都是同一套逻辑在支撑。**

### 五月：辩论队的团魂

人工智能学院辩论队，在那场比赛中明明赢了，却被「黑幕」剥夺了冠军。

那天晚上，校园墙炸了。光电学院自己的学生，都在为我们发声。

> **真正的冠军，从来不是证书上印的那个名字。**

那一刻我明白了什么叫「真正的胜利」——不是那张纸，而是人心。

### 七月：游戏与人生

暑假通关了《奇异人生》，那款游戏让我体验了一段关于选择与代价的人生。

Kate跳楼的那一刻，我感受到的是愤怒——对这个世界的残酷和无能为力的愤怒。

Chloe那句「结束它」，让我理解了：**没有完美的选择，只有你必须承受的选择。**

> **每一个选择都有代价，而有些代价是我们在做出选择时完全无法预见的。**

## 遗憾与庆幸

如果要挑一个遗憾，大概是：这一年的自己，还是太急了。

急着学完这个框架，急着掌握那门语言，急着证明自己不比别人差。回头看，有不少时间浪费在了「赶进度」上，而非真正「理解」。

但比起遗憾，更多的是庆幸。

庆幸自己没有被焦虑吞噬，而是一步一步走到了现在。庆幸遇到了辩论队的朋友们，让大学生活有了温度。庆幸选择了编程这条路，虽然艰难，却从未后悔。

2023年的我，比2022年更成熟了一点。但学得越多，越发现自己无知。这不是谦虚，而是事实。
:::
