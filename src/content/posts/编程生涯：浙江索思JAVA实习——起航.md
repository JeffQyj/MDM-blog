## 引言：站在起跑线前

2025年2月16日晚上，我躺在宿舍床上，却怎么也睡不着。

脑中回荡的是一条来自浙江索思科技有限公司的入职通知：「明天上午9点，请准时到公司报到。」

说实话，心情很复杂。兴奋、期待、紧张、忐忑……各种情绪交织在一起，像是一杯调配不当的鸡尾酒，分不清哪一口是什么味道。

这是我人生中第一份正式的JAVA开发实习。

回想2022年9月刚踏入编程大门的时候，我连 `public static void main` 是什么都不知道。而现在，我即将以「JAVA开发实习生」的身份，走进企业。

这种感觉，就像打游戏时从新手村走出来，即将踏入一片未知的地图。

## 我期待什么

说实话，在正式入职之前，我对「实习」这件事既期待又迷茫。

**期待的，是能够接触真实的项目。**

在学校里，做项目是学习的一部分，目的是巩固知识。但在公司里，做项目是工作的一部分，目的是交付价值。这两种「做项目」的心态和体验，是完全不同的。

我期待看到自己的代码被真正使用，哪怕只是一个简单的接口文档、一段辅助开发的代码片段。当代码不再是作业，而是一种「被需要」的存在时，那种感觉应该会很不一样吧。

**期待的，是能够学会团队协作。**

学校的课程设计虽然也有小组合作，但往往流于形式——分工明确却沟通匮乏，代码风格五花八门，最后勉强拼凑出一个能跑的项目。我听说在公司里，代码评审、每日站会、版本管理都是有一套规范流程的。我期待亲眼看看、亲手试试，这套「规范」究竟是怎么运转的。

**期待的，是能够认识真正的「开发者」。**

学校里的老师负责「教」，而公司的前辈负责「带」。这两者有本质区别——老师教你是因为这是他的职责，而前辈带你往往意味着他要额外付出时间和精力。我期待遇到愿意分享经验、愿意指出我问题的好前辈，也期待从他们身上学到课堂上学不到的东西。

## 我忐忑什么

有期待，自然也有忐忑。

**第一个忐忑，是怕自己「什么都不会」。**

在学校里，我自认为学得还不错——SSM框架、Redis缓存、MySQL数据库，都能说得头头是道。但仔细想想，这些「会」到底有多少是真正的「会」？恐怕要打一个大大的问号。

我知道，在真正的企业开发中，光会 CRUD 是不够的。光会调用API是不够的。光会写业务逻辑是不够的。企业开发是一套完整的工程体系，从需求分析到架构设计，从代码规范到部署运维，每一个环节都有讲究。

我忐忑的是，当真正进入开发环境，面对真实的业务场景，我能否快速适应？能否不再只是一个「纸上谈兵」的学生？

**第二个忐忑，是怕自己「问不出问题」。**

听起来很奇怪对吧？但这确实是我的担忧之一。在学校习惯了「老师讲、学生听」的模式，有问题时老师会主动引导。但在公司，前辈们各自忙碌，不可能像老师一样时刻关注你。

我听说，在公司里「会提问」是一种很重要的能力。问得太简单，显得不动脑子；问得太复杂，前辈可能也没时间解答。如何在充分思考后，问出有价值的问题，是我需要学习的第一课。

**第三个忐忑，是怕自己「心态转不过来」。**

学生时代的思维是「学习为主」，工作中是「交付为主」。这两种思维的切换，不是简单的一句话就能完成的。我怕自己还停留在「完成作业」的心态里，而没有真正意识到「承担责任」的意义。

:::note
这些忐忑，并不是因为我不自信，而是因为我对自己有期待。正因为想要做好，所以才会紧张。这份紧张，或许正是认真对待这份实习的证明。
:::

## 我想学到什么

既然是实习，我自然希望能够带走些什么。

**最想学到的，是企业开发的「规范」。**

在学校里，代码只要能跑就行。但在公司里，代码不仅要能跑，还要「别人能看懂」「便于维护」「符合团队风格」。

Git的分支策略怎么用？接口文档怎么写？Code Review看什么？这些在学校里很少被系统教授的东西，正是我最想补足的短板。

**其次想学到的，是「解决问题」的思路。**

学校里的问题往往有标准答案，或者至少有一个最优解。但工作中的问题很少这么简单——有时候不是「怎么做」，而是「该做什么」；有时候不是技术问题，而是沟通问题。

我期待学会如何在信息不完整的情况下做出判断，如何在多方利益中寻找平衡，如何把一个模糊的需求转化为可执行的方案。

**最后想学到的，是职场人的「态度」。**

我听说，真正优秀的开发者不只是技术过硬，更重要的是态度靠谱——做事认真负责、主动沟通反馈、遇到问题不推诿。这些看似简单的品质，才是区分「能用」和「好用」的关键。

## 那个夜晚

想起2022年那个第一次写出 Hello World 的下午，我曾以为自己落后了太多。两年半后的今天，我即将以实习生的身份走进一家科技公司。虽然只是辅助开发，虽然做的都是「边边角角」的工作，但这是真正的「实战」。

没有考试，没有评分，只有实实在在的交付。

当时我不知道接下来会发生什么。不知道会遇到什么样的前辈，不知道能参与什么样的项目，不知道这段经历会把我变成什么样子。

但我只知道一件事：

**这趟旅程，我已经准备好起航了。**
