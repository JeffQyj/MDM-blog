## 站在终点的回望

2022年12月31日，又一年走到尽头。

如果要用一个词形容我的2022年，大概是「转折」。从高中到大学，从题海到代码，从被动安排到主动选择——这一年的变化比过去任何一年都要剧烈。

十八年的人生里，我从未像今年这般真切地感受到：自己正在掌舵。

:::note
这篇文章不求面面俱到，只想记录那些真正塑造了我的瞬间。
:::

## 编程之路：稚嫩，但已启程

九月中下旬，我坐在工作室机房内，第一次听到了「前端三件套」这个词。

那时候的我，连 HTML 和 CSS 是什么都不清楚。于是从最基础的标签开始，我一步一步地摸索着网页的构建方式。

```html
<!DOCTYPE html>
<html>
<head>
    <title>我的第一个网页</title>
</head>
<body>
    <h1>Hello, World!</h1>
</body>
</html>
```

这个文件是我写的第一个网页。虽然现在看来简单得可笑，但彼时的成就感却无比真实。

之后的几个月里，技术栈在扩展：JavaScript 让我理解了交互，CSS 让我学会了布局，而 TypeScript 则让我第一次认识到类型系统的重要性。与此同时，C 语言的学习也在并行进行——从变量到函数，从数组到指针，每一个新的概念都在打开一扇门。

:::note
回头看那段学习时光，最珍贵的不是学会了多少语法，而是建立了「自学」的能力。老师只是领进门，真正走下去，靠的是自己。
:::

年末的 C 语言期末编程分班考，是这一年技术学习的一个小小注脚。通过了考试，意味着得到了认可；但更让我开心的是，这个过程中培养的逻辑思维，正在一点点内化为本能。

## 辩论队：推翻那个杠精的自己

说来讽刺。

高中时期的我，是朋友们眼中的「本质杠精」。凡事都要争个对错，遇到不同意见第一反应是反驳而非理解。这种性格让我在人际交往中吃过不少亏，却始终不自知。

辩论队的经历，像一面镜子，照出了我曾经的幼稚。

第一次被裁判认可表达能力时，我以为是天赋。后来才明白，那不过是辩论训练带来的副产品——学会倾听、学会组织逻辑、学会在反驳之前先理解对方。

更重要的是，辩论让我意识到：**真正的逻辑，不是让对方哑口无言，而是让双方都能看清问题的本质。**

:::warning
推翻一个旧自己从来不是一蹴而就的。辩论队教会我的，是思维方式的重塑——这比学会任何一项技能都更漫长，也更有价值。
:::

## 那些散落的收获

如果把2022年比作一幅拼图，编程和辩论是其中最显眼的两块，但还有许多碎片同样重要：

| 碎片             | 收获                       |
| -------------- | ------------------------ |
| **人工智能导论期末报告** | 三千字的写作，训练了我梳理和表达概念的能力    |
| **网页设计大作业**    | 第一次完整完成一个项目，从设计到实现的全流程体验 |
| **辩论队的朋友们**    | 一群志同道合的伙伴，让大学生活有了温度      |

这些碎片看似零散，却共同构成了我对「大学应该怎么过」的理解。

## 遗憾与庆幸

如果要挑一个遗憾，大概是：这一年的自己，还是太急了。

急着学完这个框架，急着掌握那门语言，急着证明自己不比别人差。回头看，有不少时间浪费在了无谓的焦虑上，而非真正的学习。

但比起遗憾，更多的是庆幸。

庆幸自己没有被焦虑吞噬，而是坚持走到了现在。庆幸遇到了辩论队的朋友们，让成长不再孤单。庆幸选择了编程这条路，虽然艰难，却从未后悔。

:::note
2022年的我，稚嫩、急躁、还有太多不足。但稚嫩意味着还有成长空间，急躁是年轻人的通病，而承认不足，是改变的第一步。
:::

***

*2022年的最后一天，记下这句话：虽有曲折，但没误入歧途。*

*2023，愿保持热爱，继续前行。*
